```markdown

3. 给出最终答案

...
解决方案2：
...

...
```

```markdown
3. 可能的结果
4. 选择理由

思考路径1：
...
```

## 最佳实践

### Prompt设计原则

1. **清晰明确**：避免模糊的表述
2. **提供上下文**：给出足够的背景信息
3. **指定格式**：明确输出格式要求
4. **示例引导**：使用示例引导模型
5. **迭代优化**：根据结果持续改进

### 常见问题解决

**问题1**：输出不符合预期
- 解决：增加示例，明确格式要求

**问题2**：回答过于简略
- 解决：要求详细解释，提供推理过程

**问题3**：产生幻觉内容
- 解决：限制使用提供的信息，要求引用来源

**问题4**：代码有错误
- 解决：要求测试代码，提供运行示例
